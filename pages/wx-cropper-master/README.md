# wx-cropper
微信小程序  图片裁剪工具，简单易用

QQ交流群： 424418160

### 项目需求
在做微信小程序的时候有个图片上传之前裁剪的需求，找过一些github中的项目，都不太理想，主要是没有办法自定义宽高，于是自己研究了一下，做了一个简单的图片裁剪效果

### 使用方法
项目里面直接放的该功能的文件，将这个文件夹复制到小程序项目中，在app.json中启动该页面即可看到效果


### 注意：这里面的图片在你们的项目中可能会无法使用，在测试的话 可以用自己本地或者你们项目服务器下的图片地址
